## Q1. 문자열 바꾸기 (split와 join 함수 사용)

```python
a = "a:b:c:d" #문제에서 주어진 문자열
a = a.split(':') # ':' 기준으로 문자를 나누어 리스트로 반환
a = "#".join(a) # 리스트를 문자열로 만드는데, 문자 사이를 "#"으로 이어줌
print(a)
```

🤩 **1차 결과** : a#b#c#d

## Q6. 피보나치 함수 (입력 받은 정수 n 이하까지의 피보나치 수열을 출력)

```python
n = int(input()) # 입력 13
a = 0
b = 1
c = 0

if n>0:
    print(a, end=", ")
    print(b, end="")
    while c < n:
        c = a + b
        print(", "+str(c), end="")
        a = b
        b = c    
else:
    print(0)
```

🤩 **1차 결과** : 0, 1, 1, 2, 3, 5, 8, 13
